# IPFS link

A reference to a content-addressed file on IPFS, held as an `ipfs://<cid>` string. `format: ipfs` tells an editor to render it as a file reference — a pill you can open, with upload/paste to set it — not a plain text box.


This document describes the **hypermedia-ipfs** type — a Hypermedia Network blob schema. Its formal schema is attached (the `schemaDefinition` in this document's metadata), so the app can show it and create values of this type.

## Shape

Kind: `string`.
