# Children type

How a block's children are laid out: Group (default), Ordered, Unordered, Blockquote, or Grid.


This document describes the **hypermedia-children-type** type — a Hypermedia Network blob schema. Its formal schema is attached (the `schemaDefinition` in this document's metadata), so the app can show it and create values of this type.

## Shape

Kind: `string`. One of: `Group`, `Ordered`, `Unordered`, `Blockquote`, `Grid`.
