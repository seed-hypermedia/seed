# Embed view

How an Embed block renders its target: Content, Card, Comments, or Link.


This document describes the **hypermedia-embed-view** type — a Hypermedia Network blob schema. Its formal schema is attached (the `schemaDefinition` in this document's metadata), so the app can show it and create values of this type.

## Shape

Kind: `string`. One of: `Content`, `Card`, `Comments`, `Link`.
