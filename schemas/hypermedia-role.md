# Role

A capability role: WRITER (edit a document) or AGENT (act on behalf of an account).


This document describes the **hypermedia-role** type — a Hypermedia Network blob schema. Its formal schema is attached (the `schemaDefinition` in this document's metadata), so the app can show it and create values of this type.

## Shape

Kind: `string`. One of: `WRITER`, `AGENT`.
